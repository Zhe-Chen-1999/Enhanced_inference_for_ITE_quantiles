# Enhanced_inference_for_ITE_quantiles

This project contains the supporting R code for “Enhanced inference for distributions and quantiles of individual treatment effects in various experiments”. Specifically, it includes R code for simulation studies mentioned in Section 6.1 and Appendix A3, the R code for real data analysis in Section 6.2 and 6.3, and the R code for conducting inference for average treatment effects on a binary outcome in Appendix A1.

### Key Scripts
- **`A1_Inference_for_average_treatment_effects_on_a_binary_outcome.R`**: Code to conduct inference for average treatment effects on a binary outcome in Appendix A1.
- **`A3_simulation_studies`**: R code for simulation studies mentioned in Section 6.1 and Appendix A3.
- **`helper_functions.R`**: Contains utility functions for statistical analysis.
- **`Sec6.2_Evaluating_the_effectiveness_of_professional_development`**: Real data analysis in Section 6.2.
- **`Sec6.3_Effect_of_smoking_on_the_blood_cadmium_level`**: Real data analysis in Section 6.3.
